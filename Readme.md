- Project: Scientific Lab - VR Rig Simulator Demo
- Description: Demonstrates a VR rig simulator setup and basic interactions in Unity.
- Requirements: Unity 2020.3+ (LTS) or newer; XR Interaction Toolkit; target VR headset runtime.
- Setup: Open the project, import required XR packages, add/configure the VR rig prefab in the demo scene, assign input bindings.
- Usage: Enter Play mode or build to target headset to test locomotion, head/hand tracking, and sample object interactions.
- Notes: Intended as a minimal starting point for prototyping VR lab experiments; extend scenes and scripts as needed.
